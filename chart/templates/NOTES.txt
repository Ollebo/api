ollebo-api deployed.

Public URL:
  https://{{ .Values.ingress.host }}/

Verify health from inside the cluster:
  kubectl -n {{ .Release.Namespace }} port-forward svc/{{ .Release.Name }} 8080:8080
  curl -i http://localhost:8080/healthz   # liveness
  curl -i http://localhost:8080/readyz    # readiness (checks Postgres)
  curl    http://localhost:8080/metrics   # Prometheus text format

Verify pods, HPA, Ingress:
  kubectl -n {{ .Release.Namespace }} get pods,hpa,ingress,pdb -l app=ollebo-api

External dependencies that must exist in namespace {{ .Release.Namespace }}:
  - Secret  ollebo-app  (key: password)
  - Service ollebo-rw   (Postgres/PostGIS)
  - Service nats        (NATS JetStream, port 4222)
